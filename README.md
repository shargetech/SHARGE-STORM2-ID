# 闪极超级移动电源 STORM2 外观设计

这是一套专为 DIY 爱好者和工业设计师准备的移动电源外壳及结构开源方案。本设计采用了极具机械感的透明外壳方案，并结合铝合金骨架提高整体强度和散热能力。

## 项目包含内容

本仓库提供了实现该设计所需的完整工程文件：

- 3D 模型: 通用格式 .x_t。
- CMF 说明: 详细记录了表面处理（Texture）、材质（Material）和颜色（Color）的工业要求。
- 2D 图纸: 关键装配位尺寸及公差说明（PDF）。

## 硬件规格

- **适配电芯**: 18650*8，建议使用三星、长虹三杰等品牌的 18650 电芯
- **接口开孔**: USB-C x 1, USB-A x 1, DC x 1，按键 x1
- **屏幕开孔**: 适配  1.14寸 IPS 屏，分辨率

# Shargeek Storm2 Power Bank Industrial Design (ID Design)

This is an open-source housing and structural solution for the Shargeek Storm2 Power Bank, specifically designed for **DIY enthusiasts** and **Industrial Designers**. The design features a high-tech **transparent aesthetic** combined with an **aluminum alloy internal frame** to enhance both structural rigidity and thermal dissipation.

## Project Contents

This repository provides the complete engineering files required to replicate or modify the design:

- **3D Models:** Provided in universal `.x_t` format.
- **CMF Guide:** Detailed documentation of industrial requirements for **C**olor, **M**aterial, and **F**inish (Texture).
- **2D Drawings:** PDF files specifying critical assembly dimensions and tolerances.

## Hardware Specifications

- **Battery Cells:** Designed for **8 x 18650** cells (High-quality brands like Samsung or Lishen/Sunpower are recommended).
- **Port Cutouts:** USB-C x 1, USB-A x 1, DC x 1, Physical Button x 1.
- **Display Cutout:** Optimized for a 1.14-inch IPS screen.
